# FullStackApp-with-SpringBoot-JWT-MySQL-React

- `fullstackapp-front`: A front-end built using React.js and Bootstrap 5.
- `fullstackapp-backend`: A back-end REST API built with Spring Boot, JWT, and MySQL.
